ServiceNow CMDB
Configuration Management · BOOST² Capabilities + Applications
Maps ServiceNow Configuration Items (CIs) to BOOST² Business & Technology Capability entities and Technology/Application nodes. Enables live sync of your application portfolio and infrastructure into the EA tool — dispositions (Enhance/Maintain/Retire/New) included.
// Claude-generated stub — Team 02 & 06 own this
export async function syncCMDBCapabilities() {
  const client = new ServiceNowClient({
    instance: 'nationalgrid.service-now.com',
    auth: oauthToken(process.env.SN_TOKEN)
  });
  const cis = await client.table('cmdb_ci_appl').query({
    fields: ['name','u_capability_ref','u_disposition']
  });
  return cis.map(ci => toBoost2Entity(ci, 'Application'));
}

ServiceNow SPM
Strategic Portfolio Mgmt · BOOST² Roadmap + Capability Needs
Pulls Demands, Epics, and Portfolio items from ServiceNow Strategic Portfolio Management and maps them to BOOST² Project/Program/Initiative nodes and Capability Needs. Closing a Capability Need in SPM reflects instantly in the EA tool's gap tracker.
// Claude-generated stub — Team 07 owns this
export async function syncSPMInitiatives() {
  const demands = await spmClient.getDemands({
    state: ['Approved', 'In Progress'],
    fields: ['number','short_description',
             'u_capability_need_ref','planned_start_date']
  });
  return demands.map(d => ({
    type: 'Initiative',
    closesCapabilityNeed: d.u_capability_need_ref,
    boost2Id: uuid(d.number)
  }));
}
GET/api/sn_spm/demandDemands → Initiatives
GET/api/sn_spm/portfolio_itemPortfolio → Roadmap
POST/api/sn_spm/demand/{id}Update Capability Need status
Jira Align
SAFe Portfolio · BOOST² Initiatives + Value Streams
Maps Jira Align Epics, Capabilities, and Value Streams to their BOOST² counterparts. Pulls strategic themes as input to Strategic Business Outcomes. Bi-directional: BOOST² Capability Needs can generate Jira Align Epics automatically via Claude-generated workflow.
// Claude-generated stub — Teams 01 & 07 own this
export async function syncJiraAlignValueStreams() {
  const vStreams = await jiraAlign.get('/value-streams');
  const epics = await jiraAlign.get('/epics', {
    params: { programId: PROGRAM_ID, state: 'active' }
  });
  return {
    valueStreams: vStreams.map(toBoost2ValueStream),
    initiatives: epics.map(toBoost2Initiative)
  };
}
